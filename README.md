Clone the repo:

Bash

cd Acheivers 2026
Install dependencies:

Bash

npm install
Setup Environment Variables: Create a .env.local file in the root folder and add your key:

Plaintext

VITE_GEMINI_API_KEY=your_actual_key_here
Run the app:

Bash
